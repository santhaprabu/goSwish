## üöÄ Quick Start Guide

### Day 1: Initialize Project

**Using Expo (Recommended)**:
```bash
# Install Expo CLI
npm install -g expo-cli

# Create new project
npx create-expo-app GoSwishMobile --template blank

# Navigate to project
cd GoSwishMobile

# Install dependencies
npm install @react-navigation/native @react-navigation/stack
npm install firebase
npm install react-native-paper

# Start development
npm start
```

**Using Bare React Native**:
```bash
# Create new project
npx react-native init GoSwishMobile

# Navigate to project
cd GoSwishMobile

# Install dependencies
npm install @react-navigation/native @react-navigation/stack
npm install firebase

# Start Metro
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```
